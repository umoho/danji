## 2026-06-24

### 新增：真空二极管模型

为模拟整流管行为，新增 `tube/diode.rs`，基于 Child-Langmuir 空间电荷限制定律：

```
I = K × V^γ     当 V > 0
I = 0            当 V ≤ 0
```

内置管型参数：

| 管型 | K | γ | 典型应用 |
|------|-----|-----|---------|
| 5AR4/GZ34 | 0.005 | 1.5 | 音频功放整流 |
| 5U4G | 0.003 | 1.5 | 大电流整流 |
| 6X4 | 0.002 | 1.5 | 小信号整流 |
| EZ81 | 0.004 | 1.5 | 前级整流 |

二极管作为 2 端子非线性元件集成到 MNA 求解器中，其雅可比元素为 `gd = dI/dV`。

**测试**: 5 个单元测试覆盖反向截止、正向导通、伏安特性、电导正定性、多管对比。

### 新增：电源滤波示例

`examples/power_supply.rs` — 模拟真空管整流电源：

```
AC 300V/60Hz → 5AR4 → 47µF → 220kΩ(负载) → GND
```

结果：
- B+ 输出: 297V DC (经整流管压降约 3V)
- 纹波: 0.2V p-p (47µF 在 60Hz 半波整流下的滤波效果)
- 负载电流: 1.35mA (297V/220kΩ)

### 新增：频响测量示例

`examples/tone_control.rs` — 测量 12AX7 单管放大器在不同配置下的频率响应：

| 配置 | 20Hz | 100Hz | 1kHz | 10kHz |
|------|------|-------|------|-------|
| 标准 (22µF 旁路) | +40.0dB | +35.7dB | +35.8dB | +35.8dB |
| 无旁路 | +30.1dB | +30.1dB | +30.1dB | +30.1dB |
| 小旁路 (0.22µF) | +30.1dB | +30.2dB | +33.9dB | +35.5dB |

- 标准配置：低频略增益提升（20Hz 处 +40dB vs 1kHz 处 +35.8dB），由于旁路电容与阴极电阻形成的时间常数在低频产生相位/增益变化
- 无旁路：增益约降 5.7dB，全频段平坦（局部反馈降低增益但展宽频响）
- 小旁路：呈现高通特性，从 1kHz (33.9dB) 下降到 100Hz (30.2dB)

### API 变更

`Simulator::new` 新增第三个参数 `diode_params: Vec<DiodeParams>`：

```rust
// 无二极管
Simulator::new(config, triode_params, vec![])
// 有二极管
Simulator::new(config, triode_params, vec![DiodeParams::new_5ar4()])
```

`DiodeParams` 通过 `api.rs` 公开导出。
