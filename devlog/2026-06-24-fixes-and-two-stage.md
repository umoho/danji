## 2026-06-24

### 关键 Bug 修复

实现过程中发现并修复了以下关键问题：

#### 1. 后向欧拉电容 MNA 符号错误

**症状**: 电容无法正确充放电，耦合电容的直流偏置不正确，导致双级电路瞬态求解发散。

**根因**: `stamp_capacitor` 中历史电流源符号反转。

```
// 错误（原始代码）:
i[a] += gc * (v_b - v_a);  // 历史电流进入节点 a

// 正确（修复后）:
i[a] += gc * (v_a - v_b);  // 历史电流进入节点 a
```

根据 MNA 理论，电容的伴随模型为电导 `Gc = C/h` 并联历史电流源 `I_hist = Gc * V_cap[n-1]`。对于节点 a（电容正极），历史电流源应注入 `+Gc * (Va_prev - Vb_prev)`，而非 `+Gc * (Vb_prev - Va_prev)`。

**影响**: 修复后单管增益从 72x 变为 62x（更接近理论值 58x）。

#### 2. 矩阵条件数过大

**症状**: 双管 DC 求解发散，即使使用源步进法也失败。

**根因**: 电压源电导使用 1e12，与电阻电导 1e-5 相差 17 个数量级，导致矩阵严重病态。

**修复**: 
- 引入 `VSRC_G = 1e6` 替代硬编码的 1e12
- 降低电压源电导后，双管 DC 求解器收敛（20 步源步进）

#### 3. 数值微分精度不足

**症状**: 在小电压区域，雅可比矩阵元素 gp、gm 为零，导致牛顿迭代停滞。

**根因**: 固定微分步长 1e-6 在小电压下（如 Vpk = 1e-6）无法产生可测量的电流变化。

**修复**: 自适应步长 `eps = max(1e-6, |x| * 1e-4)`

### 双级预放实现

采用**独立级联**架构（非直接 MNA 耦合）：

```
Stage 1 (12AX7) → 耦合电容 HPF → Stage 2 (12AX7)
```

- 每级独立求解 MNA（避免耦合电容 BE 电导导致的数值问题）
- 级间耦合使用指数平滑 RC 滤波器: `Vout = Vcap + alpha * (Vin - Vcap)`
- 预热 3000 样本让电路达到静态工作点

**结果**:
- 总增益: 9022x (79.1 dB)
- 每级增益: ~62x
- 实时比: 14.95x (14.95:1)

### 当前局限

- **双级直接 MNA 耦合不收敛**: 22nF 耦合电容在 44.1kHz 采样率下，BE 等效电导 Gc = C/h = 0.00097 S (≈1031Ω)，远大于栅极电阻 1MΩ，导致直流通路错误。目前使用独立级联架构绕过此问题。
- **电容初值无关**: `cap.v_prev` 字段虽被更新但不被求解器使用。求解器使用自己的 `v_prev` 数组。
